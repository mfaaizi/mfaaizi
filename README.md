## 🜁 Featured Projects

<details>
<summary><b>🟣 MeetingProxy — AI-Powered Autonomous Meeting Avatar System</b></summary>
<br/>

An AI-powered digital avatar system that autonomously attends and participates in Google Meet sessions on a user's behalf — combining real-time speech, video avatar rendering, and conversational AI into a single automated pipeline.

| Category | Detail |
|---|---|
| **Stack** | Flask, React (Vite), GPT-4o-mini, ElevenLabs, OpenAI Whisper, Selenium, OBS Studio, D-ID API, Firebase, Cloudinary, VB-Audio Cable |
| **Scale** | Full-stack system with 18 frontend page components, real-time audio/video pipeline |
| **Performance** | Real-time transcription and response generation with low-latency avatar sync |
| **Security** | Firebase-authenticated sessions, resolved API key exposure hardening |
| **Impact** | Automates live meeting attendance and participation end-to-end |
| **Repository** | `Private / Available on Request` |

Built as a Final Year Project, MeetingProxy integrates automated browser control (Selenium) with a digital avatar rendering pipeline (D-ID API) and virtual audio routing (VB-Audio Cable + OBS Studio) to simulate authentic meeting presence. The system was extended with a **FastMCP server layer**, exposing the entire application as agent-callable MCP tools. Full FYP documentation, a 38-slide technical presentation, architecture diagrams, and structured test cases were produced alongside the build.

</details>

The project included a full UI redesign with a custom maroon color system, Firebase infrastructure built from scratch, and a 14-slide entrepreneurship pitch deck for commercial positioning.
